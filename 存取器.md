```js
var s = {
  // 普通属性
  x: 1,
  y: 1,
  
  get r() {
    return Math.sqrt(this.x ** 2 + this.y ** 2);
  },
  
  set r(val) {
    const ratio = val / this.r;
    this.x *= ratio;
    this.y *= ratio;
  }
}
```

数据属性 的4个特性
1. 值 (value)
2. 可写性 (writable)
3. 可枚举 (enumerable)
4. 可配置 (configurable)

存取器属性的4个特性
1. 取 (get)
2. 写入 (set)
3. 可枚举 (enumerable)
4. 可配置 (configurable)
